# Git2Markdown
### Easily feed an LLM a text summary of any project or repository folder on your machine.

Simply drag any repo folder into the app and specify the file types to scan and what files and folder to ignore (i.e. .gitignore, node_modules, etc). 
Export and send that text file to your favorite LLM.
